Pizza Shop Simulator

This is a useless pizza shop simulator, created as a final project to Java course, so frontend part is very simple

What can I do with this app?

• Go to 'Restaurants' -> 'Catalog' Tab to see all available restaurants

• Go to 'Pizzas' -> 'Catalog' Tab to see a whole pizza menu

• Search for particular restaurant/menu going to 'Restaurants'/'Pizzas' -> 'Search' Tab

• Log in to make an order ('Login' Tab)

• To create a new profile, go to 'Login' -> 'New customer?' Tab

• Click on restaurant/pizza name to see a detailed info about it

• Click on 'Add to Card' button to add it to your order

• Your current order is on 'Card' Tab

• There you can also make an order by clicking on special button

NOTICE that after the order confirmation your Card will be empty

• Add/Delete restaurants/pizzas in Administrator tools Tab

NOTICE that this feature is for admins only

*but, actually, you can become an Admin after Login with 'admin@pizza.com' - 'Passw0rd!' credentials
